# GoBangumi

使用Golang编写的全自动追番工具，相较于[AutoBangumi](https://github.com/EstrellaXD/Auto_Bangumi) 简化部署和使用，以及更方便的模块化扩展

## 目的
- 简化部署和使用，以及模块化扩展
- 学习
